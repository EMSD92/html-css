Tag code + tag pre

Para digitar codigos na pagina html, com suas formatações e tabulações, deve-se usar a tag pre e também a tag code

código inserido somente com a tag code

num = int (input('Digite um número')) if num %2 ==0; print(f'O número {num} é PAR') else: print('f'O numero {num} é IMPAR') printf('Fim do programa')

código inserido com a tag code e a tag pre

        
num = int (input('Digite um número'))
if num %2 ==0;
    print(f'O número {num} é PAR')
else:
    print('f'O numero {num} é IMPAR')
printf('Fim do programa')
